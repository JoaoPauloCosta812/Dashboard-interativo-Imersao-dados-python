# Dashboard-interativo---Imersao-dados-python
Dashboard interativo criado durante a aula 04 da imersão de dados python da Alura

Durante a imersão trabalhamos uma base de dados retirada do Kaggle pela alura, que trazia a informação de salarios de cargos na area de dados.

Tratamos os dados nas 2 primeias aulas, fazendo uso das bibliotecas Pandas e Numpy e depois seguimos para a visualização dos dados, na aula 3 com
as bibliotecas Seaborn, Matplotlib e a Plotly, criando diferentes tipos de graficos para fazer a analise dos dados, onde fizemos um desafio da criação de
um grafico referente ao salario de cientistas de dados em cada país (a solução desde desafio também está disponivel nos meus repositorios aqui no GitHub).
E por fim na aula 4 fizemos a criação desse Dashbord interativo, que traz uma visualiação mais direta e pratica para analise dos dados trabalhados na imersão;
o mesmo foi criado a partir da biblioteca Streamlit, que também foi usada no compartilhamento do app, diponivel do link: https://dashboard-salarios-na-area-de-dados-imersao-dados-python.streamlit.app

O dashbord traz a infromações de Salários na area de dados, incluindo diferentes cargos e países, fique a vontade para explorar os dados e dar uma olhada no codigo criado aqui.
